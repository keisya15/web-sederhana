Pekerjaan untuk bidang Telekomunikasi

Contoh profesi:

Menjadi seorang IT ( Information Teknologi ) berkaitan dengan penggunaan komputer untuk mengumpulkan, menyimpan, mengamankan, dan mengelola informasi.
System Engginer ( Teknisi Sistem ) merancang dan mengimplementasikan software untuk berbagai informasi dan aplikasi berbasis teknologi serta prosedur pengujian untuk sistem internal.
Network Analisyt ( analis sistem ) melakukan analisis, merancang, dan mengimplementasikan sistem.
System Administrator melakukan pengelolaan sistem komputer, baik komputer client, komputer server, hingga perangkat-perangkat network seperti printer dan scanner.
Network Manager ( Pengelola Jaringan ) Mendesain dan membangun infrastruktur jaringan computer dan perangkat keras computer. Melakukan installasi, troubleshooting dan perbaikan infrastruktur jaringan computer dan perangkat keras. Memonitoring dan memelihara infrastruktur jaringan komputer dan pusat data agar dapat berfungsi dan berjalan dengan baik.
IT architec ( arsitek IT ) Mengembangkan dan menerapkan strategi data perusahaan secara keseluruhan yang sejalan dengan proses bisnis.
Database Administrators ( pengelola basis data ) admin dalam pengeloaan sistem komputer bekerja secara khusus untuk melakukan perawatan (Maintenace) ruang lingkup database serta mengarahkan semua kegiatan keamanan data.
